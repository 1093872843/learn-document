# 快速开始使用 openAI SDK

[deepseel developer platform](https://platform.deepseek.com/api_keys)

## 准备工作

1. 注册账号
2. 获取[api key](https://platform.deepseek.com/api_keys)
3. 将获取到的变量存入环境变量中

```shell
// linux
export DEEPSEEK_API_KEY="your_api_key_here"
// window
setx DEEPSEEK_API_KEY "your_api_key_here" 
```
